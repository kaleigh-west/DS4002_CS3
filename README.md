# How to Obtain the Raw Data Set 
To obtain the data set for this case study, go to this link: https://www.kaggle.com/datasets/clmentbisaillon/fake-and-real-news-dataset?resource=download. From this link, click on dowload to retrieve the data files (this will dowload a zip file). When you un-zip the file, you will see two CSV files: Fake.csv (contains the fake news stories) and True.csv (contains the real news stories).
