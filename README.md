### How to Obtain the Data Set 
You will be using the ISOT Fake News Dataset for this case study. To obtain dataset, go to this link: https://www.kaggle.com/datasets/clmentbisaillon/fake-and-real-news-dataset?resource=download. From this link, click on dowload to retrieve the data files (this will dowload a zip file). When you un-zip the file, you will see two CSV files: Fake.csv (contains the fake news stories) and True.csv (contains the real news stories).
